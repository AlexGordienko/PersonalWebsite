Projects



Knowtworthy


Knowtworthy is an automated note taking system which takes in lecture or meeting videos and returns both a set of detailed notes as well as a brief summary of the contents. Using state-of-the-art machine learning and powerful heuristics, Knowtworthy aims to achieve simple and scalable access to quality educational content.

Giftora


Giftora is an app for created curated gifts on demand. Built for Android and IOS on Ionic3 with Sails on Node, Giftora takes the stress out of gift giving and instead provides an excellent gift giving experience for both the giver and the receiver. You can try it now!

Appleby Wellness


Appleby Wellness is a landing page built in HTML, CSS and JavaScript (with JQuery) for a contractor. On top of building a landing page I provided SEO services to boost the company page's throughput and increase revenue.

OpenMaze


OpenMaze is an open source software package that provides and easy an intuitive platform to quickly set up and run cognitive tasks in a robust 3D without the need for programming expertise. The app was built in Unity and coded in C#. The landing page is standard static HTML, CSS and JS.

MEAN Stack Web Application


During my time at Explore Canada, a startup in Mississauga, I worked with one other developer to build both a landing page as well as an intelligent client processing form application to both streamline and automate as much of the startup's work as possible while constructing a professional and functional face of the company. As a result of this work the startup began to receive a heightened number of quality clients, illustrating the overwhelmingly positive effect our work had on the company.

June 2017 - December 2017

    At Explore Canada I was given the opportunity to completely redesign the company website from the ground up. Bringing my experience from previous projects built with the MEAN Stack as well as my cooperative and adaptable nature, Manav Patel and I constructed not only the company landing page but also a complex form processing app equipped with powerful logic and user friendly content.

     Using Angular 4 and the Material 2 library we worked to build a thoroughly modern and impressive material design app while supporting the front-end with powerful and functional back-end. With Express.js, Node.js and Mongo DB we assembled a multifaceted back-end that utilized the latest in analytics, validation, mail api integration and accessible and scalable storage solutions. Furthermore the product that was delivered was not only optimized for speed and scalability but also optimized for SEO and was fully documented ensuring that future development could begin seamlessly. Currently I am continuing to optimize the site while adding new features and maintaining it.

     While working at Explore Canada I learned a great deal about MEAN stack development, but among the most rewarding experiences of working at this start up was being able to focus on a strong product and work together towards a common goal. Constant back and forth with the CEO allowed for the implementation of an organizational structure that benefited both parties. Furthermore utilizing Agile Development practices (specifically through SCRUM) allowed the project to flow seamlessly and rapidly towards completion. Communication, hard work, teamwork and trust all played a large role in the production of the app and I left with organizational and interpersonal skills that could be applied to any organization no matter how big or small.
